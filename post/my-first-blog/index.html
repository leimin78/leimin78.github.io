什么是Hugo

参考wiki Hugo wiki Git仓库地址 Hugo git

安装Hugo

下载对应的编译版本后可直接运行, 本地环境需要有git，及golang

Hugo 简单使用

1.使用hugo new site xxxx 创建属于自己的blog目录 例如 hugo new site myblog

2.挑选喜欢的主题 hugo 主题库地址 Hugo theme

3.使用主题
进入hugo 站点目录找到自己心仪的主题 clone 例如: cd myblog

    git clone https://github.com/zhaohuabing/hugo-theme-cleanwhite.git themes/hugo-theme-cleanwhite

修改config.yaml 例如:

baseURL = "http://leimin78.github.io/"
languageCode = "zh-CN"
title = "开心-Coder"
theme = "hugo-theme-cleanwhite"

[outputs]
home = ["HTML", "RSS"]

创建日志

例如： hugo new post/my-first-blog.md 执行该命令后日志文件存放在content/post目录中 自行使用markdown语法编辑即可

如何上传到github ，并使用github page

1.github 创建两个仓库 仓库1 ，名称任意用于存放hugo 原始文件。 仓库2 ，命名必须为.github.io 用于存放hugo生成的静态页面文件，被github pages直接解析

仓库1推送步骤. 例如:

git init
git remote add origin https://github.com/leimin78/my-hugo.git
git add .
git commit -m "hugo 初始化"
git push origin master

仓库2推送步骤 先使用hugo命令，生成public目录 进入public目录后同仓库1步骤如下: 例如:

git init
git remote add origin https://github.com/leimin78/leimin78.github.io.git
git add .
git commit -m "hugo 初始化"
git push origin master

完成上述操作后，打开 leimin78.github.io 就可以访问你自己的blog
